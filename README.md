# MicroSnap Builder
MicroSnap Builder is a lightweight browser-based tool for creating and sharing short, visual informational slides. Itâ€™s designed for quick micro-content, letting users build animated content using simple form inputs or JSON.

## Features
- Build micro slide decks using a simple editor or JSON input  
- Live preview updates as you modify content  
- Preset layouts, positioning options, and animations  
- Export decks as standalone HTML files  
- Optional API endpoint for programmatic rendering  
- Responsive design for embedding or mobile viewing  

## Purpose
This project was created to explore efficient ways of presenting small pieces of information in a clean, interactive format. MicroSnap Builder supports quick storytelling, teaching moments, and shareable micro-learning without the overhead of full authoring tools.

## Technologies Used
- HTML, CSS, JavaScript  
- Node.js with Express (export/render API)  
- Client-side JSON rendering engine
